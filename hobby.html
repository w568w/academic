Hobby

As a computer science student, I naturally have a keen interest in everything related to computers! I've been a computer enthusiast since around 2006 and began my programming journey approximately in 2014.

My first programming language was Easy Programming Language (易语言), a Chinese-syntax-based, VisualBasic-like RAD programming language for Windows applications. Following that foundation, I explored C, C++, Java, JavaScript, and more. Upon entering university, I started exploring newer programming languages. Currently, I've become an enthusiast of Rust, Zig, and Go.

I'm a Windows devotee who has been consistently using the system from Windows 2000 through 11 since 2006. However, my technological curiosity extends beyond; as an Arch Linux advocate, I've compiled my own custom kernels and ported things across platforms. I enjoy fixing things and learning how they work, which has led me to tinker with hardware, software, and firmware alike.

Beyond programming, I'm a casual video gaming enthusiast with a diverse spectrum of interests: spanning Minecraft's sandbox genre, RTS classics like Red Alert 2, racing franchises from Need for Speed: Underground to Forza Horizon 5, and triple-A masterpieces ranging from Grand Theft Auto V to Baldur's Gate 3. I also play mobile games, spending some time on rhythm games like Cytoid, as well as gacha games like Arknights and Wuthering Waves.
